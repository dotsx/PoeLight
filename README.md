# PoeLight

PoeLight is inspired by PoeSmoother, so many features are the same. You can choose any of them for free.

## Features
- highlight / color mods
- remove fog
- remove enviro particles
- remove shadows
- remove darkness
- remove skills sounds
- remove monsters sounds
